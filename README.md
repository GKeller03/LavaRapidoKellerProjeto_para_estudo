# 🚗 Sistema de Lava Rápido (Projeto de Estudo)

Este projeto foi desenvolvido com o objetivo de praticar conceitos de programação em Java, incluindo orientação a objetos e criação de interfaces gráficas com Swing.

## 💻 Funcionalidades

- Cadastro de clientes
- Cadastro de tipos de serviço
- Simulação de agendamento
- Interface simples com menu interativo

## 🛠️ Tecnologias utilizadas

- Java
- Swing (JOptionPane)
- Programação Orientada a Objetos (POO)

## 🧱 Estrutura do projeto

O sistema está organizado em:

- `model`: classes de dados (Cliente, TipoServico)
- `view`: interfaces do sistema
- `main`: classe principal com menu de execução

## ▶️ Como executar

1. Baixe ou clone o repositório  
2. Abra no NetBeans  
3. Execute a classe `Main.java`  
4. Utilize o menu para testar as funcionalidades  

## 📌 Observações

Este projeto tem fins acadêmicos e representa um primeiro contato com desenvolvimento de sistemas em Java.
